Данный способ является самым быстрым. gsettings set org.gnome.gedit.preferences.encodings auto-detected "['UTF-8', 'WINDOWS-1251', 'CURRENT', 'ISO-8859-15', 'UTF-16']" Теперь, если вы откроете файл с кодировкой WINDOWS-1251 - он будет правильно отображаться в Gedit. (gedit исчезает из списка доступных приложений, но его можно запустить из консоли) Взято из статьи.
