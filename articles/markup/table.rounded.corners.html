Источник: http://red-team-design.com/practical-css3-tables-with-rounded-corners/ Here's the trick: while border-collapse's default value is separate, we need also to set the border-spacing to 0. 
table { *border-collapse: collapse; /* IE7 and lower */ border-spacing: 0; }
Then, we just need to round some corners 
th:first-child { border-radius: 6px 0 0 0; }
th:last-child { border-radius: 0 6px 0 0; }
th:only-child{ border-radius: 6px 6px 0 0; }
Может возникнуть проблема: в мозилле пустые клетки не имеют бордера. Лечится св-вом empty-cells: show
