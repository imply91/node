Перейти в рабочую директорию cd /opt/projects/abcp/ Логи, последние три коммита git log -3 Откатить до коммита с хэшем <commit> git revert <commit> git reset --hard <commit> Форсированная перезаливка ветки git push -f origin myBranch Обновиться с ветки (в данном случае с мастера) git pull --rebase origin master Интерактивный ребейз для склейки коммитов, где <commit> - хэш коммита, который еще не должен попадать в склейку, схлопываются коммиты, следующие за ним git rebase -i <commit> Если коммит с мержем уже создался, то избавиться от него можно ребейзом с мастера. git rebase origin/master
